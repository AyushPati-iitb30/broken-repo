# Assignment 1: Sensor Data Logger

## Description
This repository contains a simple sensor data logging program. The program simulates a simple sensor data logger written in C++. It reads sensor values and writes the results to output files.

## Output Directory
The `src/output/` directory contains the valid output files generated by the program. These are the only files that should exist in the repository as program output.
